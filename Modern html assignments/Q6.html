Html and Html5
HTML HTML5
Limited semantic elements like div and span. Relies on classes and IDs for meaning. Introduces semantic elements such as header, footer, article, section, nav, and aside for better document structure.
Limited native support for multimedia. Often relied on third-party plugins like Flash. Native support for multimedia with audio and video elements. Eliminates the need for external plugins, supporting features like streaming and offline playback.
No native support for drawing graphics directly in the browser. graphics directly in the browser. Introduces the canvas element for dynamic rendering of graphics, charts, and animations using JavaScript.
